プライバシーポリシー

アプリ名: ReminDo（リマインドゥ）
運営: NARUKAMI AKUT
最終更新: 2026年6月6日

本プライバシーポリシーは、ReminDo アプリ（以下「本アプリ」）がどのような情報を収集し、どのように使用するかを説明します。

1. 収集する情報

データの種類内容目的
メールアドレスアカウント登録時に入力したメールアドレスアカウント管理・認証
ユーザー IDFirebase が発行する匿名識別子デバイス間同期
タスクデータユーザーが入力したリストアイテム・プロファイル名アプリ機能・デバイス間同期
アプリの操作ログタスク作成・完了などの操作履歴（Firebase Analytics）アプリ改善・分析
クラッシュログアプリ異常終了時のエラー情報（Firebase Crashlytics）バグ修正・品質改善
広告識別子デバイスの広告 ID（AdMob）広告配信
カレンダーデータGoogle カレンダー連携をオンにした場合のみ、期限付きタスクの情報Google カレンダーとの同期

2. 情報の共有

収集した情報は以下の第三者サービスと共有されます。いずれも Google LLC が提供するサービスです。

  • Firebase Authentication — アカウント管理
  • Cloud Firestore — データ同期
  • Firebase Analytics — 使用状況分析
  • Firebase Crashlytics — クラッシュ報告
  • Google AdMob — 広告配信（無料プランのみ）
  • Google Calendar API — カレンダー同期（ユーザーが連携をオンにした場合のみ）

上記以外の第三者への個人情報の提供は行いません。

3. データの保存と保護

すべてのデータは HTTPS（TLS）で暗号化して転送されます。データは Google のサーバー（Firebase）に保存されます。

4. アカウントとデータの削除

アプリ内からアカウントを削除できます：
アプリを起動 → 設定 → アカウント → アカウントを削除

削除すると、Firestore 上のすべてのデータおよびアカウント情報が完全に削除されます。この操作は取り消せません。

削除のご依頼やご不明点は、下記メールアドレスまでお問い合わせください。

5. 子どものプライバシー

本アプリは家族向けに設計されており、13歳未満の子どもから意図的に個人情報を収集することはありません。保護者の管理のもとでご利用ください。

6. ポリシーの変更

本ポリシーを変更する場合は、このページを更新します。重要な変更がある場合はアプリ内でお知らせします。
